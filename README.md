# 青花瓷数字博物馆

这是一个综合性的青花瓷数字博物馆网站，提供多维度的数据可视化和分析功能。

## 功能特点

✨ **主要功能模块**：

### 🏠 首页
- 精美的封面设计，展现青花瓷文化魅力
- 导航卡片，快速访问各个功能模块
- 青花瓷主题色彩体系，统一视觉风格

### 🎨 颜色可视化
- 📸 展示所有青花瓷图片
- 🎨 显示每张图片的RGB颜色色卡
- 📊 颜色比例可视化（色块和条形图）
- 🔍 搜索功能（按ID、类型、URL搜索）
- 🎯 类型过滤
- 📈 多种排序方式
- 📊 颜色流动关系桑基图

### 🗺️ 地理分布
- 交互式地图展示青花瓷的地理分布
- 标记点显示详细信息（时期、年代、类型、尺寸等）
- 图片预览和描述信息

### 📊 数据分析
- 朝代分布统计（饼图）
- 器型分布统计（柱状图）
- 装饰元素分析（横向柱状图）
- 时间分布趋势（折线图）
- 高度分布统计
- 朝代-器型关系热力图
- 详细数据表格（支持搜索和过滤）

### 📝 文本分析
- 关键词词频分析（Top 20）
- 关键词云图可视化
- 朝代-关键词关系分析
- 器型-关键词关系分析
- 关系图谱（展示朝代、器型、关键词之间的复杂网络关系）

## 使用方法

### 🏠 本地运行（开发/测试）

#### 方法1：使用Python内置服务器（推荐）

1. 确保所有文件在同一目录下

2. 在终端中运行：
```bash
cd "/Users/kkrois/Desktop/blue_white porcelain"
python3 -m http.server 8000
```

3. 在浏览器中打开：
```
http://localhost:8000/index.html
```

#### 方法2：使用 Node.js http-server
```bash
npm install -g http-server
cd "/Users/kkrois/Desktop/blue_white porcelain"
http-server -p 8000
```

## 网站功能说明

### 搜索功能
- 在搜索框中输入关键词，可以搜索：
  - ID号
  - 类型（如：Ceramics, bottle, bowl等）
  - URL地址

### 类型过滤
- 下拉菜单中选择特定类型，只显示该类型的图片

### 排序功能
- **ID 升序/降序**：按ID号排序
- **颜色数 升序/降序**：按RGB颜色种类数量排序

### 颜色展示
每张图片卡片包含：
- 图片预览
- RGB颜色色卡（显示所有检测到的颜色）
- 颜色条形图（显示各颜色的比例）
- 每个颜色显示RGB值和百分比

## 文件结构

```
blue_white porcelain/
├── index.html                    # 网站首页
├── colors.html                   # 颜色可视化页面
├── map.html                     # 地理分布页面
├── analysis.html                # 数据分析页面
├── text_analysis.html           # 文本分析页面
├── color.csv                    # 颜色数据文件
├── location_data.json           # 地理分布数据
├── analysis_data.json           # 数据分析数据
├── text_analysis_data.json      # 文本分析数据
├── sankey_data.json             # 桑基图数据
├── Processed_Data.csv           # 处理后的完整数据
├── 生成青花瓷封面.png            # 封面图片
├── README.md                    # 说明文档
├── 部署指南.md                   # 部署指南
├── deploy.sh                    # 快速部署脚本
├── .gitignore                   # Git 忽略文件配置
│
├── scripts/                     # 开发脚本文件夹（可选）
│   ├── *.py                     # Python 数据处理脚本
│   └── *.log                    # 日志文件
│
├── docs/                        # 文档文件夹（可选）
│   ├── 设计评价与改进建议.md
│   └── index_old.html          # 历史版本
│
└── data/                        # 数据备份文件夹（可选）
    └── backup/                  # 备份数据文件
```

### 📁 文件夹说明

- **根目录**：包含网站运行所需的所有文件（HTML、数据文件、配置文件）
- **scripts/**：开发过程中使用的 Python 脚本和日志（部署时不需要）
- **docs/**：项目文档和历史文件（部署时不需要）
- **data/backup/**：数据备份文件（部署时不需要）

**注意**：部署网站时，只需要根目录的文件即可。其他文件夹的内容是可选的。

## 🚀 方法一：GitHub Pages（推荐 - 免费且简单）

GitHub Pages 是 GitHub 提供的免费静态网站托管服务，非常适合您的项目。

### 步骤：

1. **创建 GitHub 仓库**
   ```bash
   # 在项目目录中初始化 git
   cd "/Users/kkrois/Desktop/blue_white porcelain"
   git init
   git add .
   git commit -m "Initial commit: 青花瓷数字博物馆"
   ```

2. **在 GitHub 上创建新仓库**
   - 访问 https://github.com/new
   - 仓库名：`blue-white-porcelain-museum`（或您喜欢的名字）
   - 设置为 Public（GitHub Pages 免费版需要）
   - 不要初始化 README（因为您已有文件）

3. **推送代码到 GitHub**
   ```bash
   git remote add origin https://github.com/您的用户名/blue-white-porcelain-museum.git
   git branch -M main
   git push -u origin main
   ```

4. **启用 GitHub Pages**
   - 进入仓库页面
   - 点击 `Settings`（设置）
   - 左侧菜单找到 `Pages`
   - Source 选择 `Deploy from a branch`
   - Branch 选择 `main`，文件夹选择 `/ (root)`
   - 点击 `Save`

5. **访问网站**
   - 几分钟后，您的网站将在以下地址可用：
   - `https://您的用户名.github.io/blue-white-porcelain-museum/`

### 优点：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 自定义域名支持
- ✅ 版本控制集成
- ✅ 自动部署（推送代码即更新）

### 缺点：
- ⚠️ 需要公开仓库（免费版）
- ⚠️ 有带宽限制（通常足够使用）

## 🌐 方法二：Netlify（推荐 - 最简单）

Netlify 提供免费的静态网站托管，支持拖拽部署。

### 步骤：

1. **准备部署文件**
   - 确保所有文件都在项目根目录

2. **访问 Netlify**
   - 访问 https://www.netlify.com/
   - 使用 GitHub 账号登录（推荐）或邮箱注册

3. **部署方式 A：拖拽部署（最简单）**
   - 登录后，直接将整个项目文件夹拖拽到 Netlify 的部署区域
   - 等待上传完成
   - 网站会自动部署并生成一个随机域名

4. **部署方式 B：Git 集成（推荐）**
   - 点击 `Add new site` → `Import an existing project`
   - 连接您的 GitHub 账号
   - 选择仓库
   - 构建设置：
     - Build command: 留空（静态网站不需要构建）
     - Publish directory: `/` 或留空
   - 点击 `Deploy site`

5. **自定义域名（可选）**
   - 在站点设置中，可以：
     - 使用 Netlify 提供的免费子域名
     - 或绑定您自己的域名

### 优点：
- ✅ 完全免费
- ✅ 拖拽即可部署，最简单
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 自动部署（Git 推送即更新）
- ✅ 支持表单处理、函数等高级功能

### 缺点：
- ⚠️ 免费版有带宽限制（100GB/月，通常足够）

## ⚡ 方法三：Vercel（推荐 - 性能优秀）

Vercel 专注于前端部署，性能优秀。

### 步骤：

1. **访问 Vercel**
   - 访问 https://vercel.com/
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 `Add New Project`
   - 导入您的 GitHub 仓库
   - 或使用 Vercel CLI：
     ```bash
     npm i -g vercel
     cd "/Users/kkrois/Desktop/blue_white porcelain"
     vercel
     ```

3. **配置部署**
   - Framework Preset: `Other`
   - Root Directory: `./`
   - Build Command: 留空
   - Output Directory: `./`
   - 点击 `Deploy`

4. **访问网站**
   - 部署完成后会获得一个 `*.vercel.app` 域名

### 优点：
- ✅ 完全免费
- ✅ 性能优秀，全球 CDN
- ✅ 自动 HTTPS
- ✅ 自动部署
- ✅ 支持预览部署

### 缺点：
- ⚠️ 免费版有带宽限制（100GB/月）

## 🐳 方法四：Docker + 云服务器

如果您有自己的服务器，可以使用 Docker 部署。

### 步骤：

1. **创建 Dockerfile**
   ```dockerfile
   FROM nginx:alpine
   COPY . /usr/share/nginx/html
   EXPOSE 80
   CMD ["nginx", "-g", "daemon off;"]
   ```

2. **构建和运行**
   ```bash
   docker build -t porcelain-museum .
   docker run -d -p 80:80 porcelain-museum
   ```

## ⚠️ 部署注意事项

### 1. CORS 问题
- 确保使用 HTTPS（所有推荐的平台都自动提供）
- 如果遇到跨域问题，检查数据文件的路径是否正确

### 2. 路径问题
- 确保所有文件路径使用相对路径（如 `./data.json` 而不是 `/data.json`）
- 检查图片路径是否正确

### 3. 文件大小
- 大文件（如 CSV、JSON）可能影响加载速度
- 考虑压缩或使用 CDN

### 4. 浏览器兼容性
- 现代浏览器（Chrome、Firefox、Safari、Edge）都支持
- 如需支持 IE，需要额外配置
